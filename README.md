# James Malcolm Blog

This is the repo for [jamesmalcolm.me](https://jamesmalcolm.me/).

All Blog analysis/notebooks of analysis are located in a [dedicated repo](https://github.com/JamesTrick/blog_analysis).

## Local development

- Start local server: hugo serve
- Build production output: hugo --environment production --gc --minify

## Deployment

This site is deployed on Netlify.

Suggested Netlify settings:

- Build command: hugo --environment production --gc --minify
- Publish directory: public
